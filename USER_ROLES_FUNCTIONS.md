## Admin Functions

### User Management
- **Roles & Permissions** - Create, edit, and manage user roles and permissions
- **Sidebar Permissions** - Control which menu items each user can see
- **User Management** - Create, view, edit, and delete system users
- **Assign Roles** - Assign roles to users

### Student Management
- **Create Students** - Add new students to the system (with or without parents)
- **Edit Students** - Update student information
- **Delete Students** - Remove students from the system
- **View Student Details** - View comprehensive student profiles
- **Student ID Cards** - Generate and download student ID cards
- **Webcam Photo Capture** - Capture student photos for ID cards
- **Force Password Reset** - Force students to change their password
- **Import Students** - Bulk import students from file
- **Student Upgrade** - Promote students to the next class level
- **Seat Assignment** - Assign chairs and desks to students

### Teacher Management
- **Create Teachers** - Add new teachers
- **Edit Teachers** - Update teacher information
- **Delete Teachers** - Remove teachers
- **Teacher Sessions** - Manage teacher session assignments
- **Force Password Reset** - Force teachers to change passwords
- **Teacher Device Management** - Manage teacher mobile devices for attendance
- **Teacher QR Codes** - Generate/manage QR codes for attendance scanning

### Parent Management
- **Create Parents** - Add new parents
- **Edit Parents** - Update parent information
- **Delete Parents** - Remove parents
- **Force Password Reset** - Force parents to change passwords
- **Send Registration SMS** - Send parent self-registration links

### Class Management
- **Create Classes** - Add new classes/grades
- **Edit Classes** - Update class information
- **Delete Classes** - Remove classes
- **Assign Subjects to Classes** - Link subjects to specific classes

### Subject Management
- **Create Subjects** - Add new subjects
- **Edit Subjects** - Update subject details
- **Delete Subjects** - Remove subjects
- **Assign Subjects** - Assign subjects to classes
- **Bulk Unassign** - Remove multiple subject assignments
- **Onboard Subjects** - Quick subject onboarding

### Syllabus Management
- **Create Syllabus Topics** - Add curriculum topics
- **Edit Syllabus Topics** - Update topic details
- **Delete Syllabus Topics** - Remove topics
- **View Teacher Schemes** - Review teacher scheme submissions
- **View Teacher Syllabus Progress** - Monitor syllabus coverage

### Results & Assessments
- **View All Results** - Access results across all classes
- **Manage Results Status** - Control term/result period settings
- **Results Approval** - Approve or reject teacher-submitted results
- **Assessment Marks Approval** - Approve assessment marks
- **Clean Results** - Remove erroneous result entries
- **Exempt Students** - Exempt students from result requirements

### Attendance Management
- **Attendance Logbook** - View teacher attendance records
- **QR Scanner Attendance** - Scan teacher QR codes for check-in/out
- **Attendance History** - View and manage attendance records
- **Attendance Settings** - Configure attendance parameters
- **Export Attendance** - Download attendance reports
- **Student Class Attendance** - View student attendance by class

### Finance & Accounting
- **Student Payments** - Record and manage student fee payments
- **Parents Arrears** - View and manage outstanding balances
- **School Income** - Record and track school income
- **School Expenses** - Record and manage expenses
- **Financial Statements** - Generate financial reports
- **Products & POS** - Manage school store and point-of-sale
- **Inventory Management** - Track product stock levels
- **Sales History** - View sales records and receipts
- **Product Categories** - Manage product categorization

### Payroll
- **Salary Management** - Set up staff salaries
- **Generate Payroll** - Create monthly payroll
- **Approve Payroll** - Approve payroll entries
- **Mark as Paid** - Record salary payments
- **Payslips** - Generate and print payslips

### Cash Book
- **Cash Book Entries** - Record cash transactions
- **Edit Entries** - Modify cash book records
- **Cash Book Reports** - Generate cash flow reports

### Expenses Management
- **Record Expenses** - Add expense entries
- **Expense Categories** - Manage expense categories
- **Approve Expenses** - Authorize expense claims
- **Expense Reports** - Generate expense summaries

### Purchase Orders
- **Create Purchase Orders** - Initiate procurement requests
- **Approve Purchase Orders** - Authorize purchases
- **Mark as Ordered/Received** - Track order status

### Term & Fee Management

- **Fee Level Groups** - Set fees by class levels
- **Fee Structures** - Define fee amounts per category

### Timetable Management

- **Edit Timetable** - Modify schedule entries

### Library Management

### Leave Management

### Groceries Management

### Disciplinary Records

### Medical Reports

### Notifications

- **View Notifications** - See all sent notifications
- **Delete Notifications** - Remove notifications

### Events Management
- **Create Events** - Add school events
- **Edit Events** - Modify event details
- **Delete Events** - Remove events

### Newsletters

- **Edit Newsletters** - Update newsletter content
- **Publish Newsletters** - Make newsletters public
- **Delete Newsletters** - Remove newsletters

### Banner Management

- **View Applicants** - See student applications

- **Delete Applications** - Remove application records

### Staff Management
- **Create Staff** - Add non-teaching staff
- **Edit Staff** - Update staff information
- **Delete Staff** - Remove staff records

### Marking Scheme
- **View Class Assessments** - Access assessment overview
- **View Assessment Marks** - See detailed mark breakdowns

### Audit Trail
- **View Audit Logs** - Track system activities
- **Export Audit Trail** - Download audit records
- **Clear Audit Trail** - Archive old logs

### Reports
- **Financial Reports** - Income statements, balance sheets
- **Fee Reports** - Fee collection summaries
- **Expense Reports** - Expenditure analysis
- **Export Reports** - Download report data

### Settings
- **Class Formats** - Configure class naming conventions
- **Upgrade Direction** - Set class progression rules
- **Geolocation Settings** - Define school boundaries for attendance
- **SMS Settings** - Configure SMS gateway
- **Website Settings** - Customize public website
